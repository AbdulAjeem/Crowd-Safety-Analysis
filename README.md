🚨 Crowd Safety Analysis AI

A Computer Vision-based AI system that analyzes crowd images to estimate density, visualize crowd concentration using heatmaps, and generate safety recommendations through an interactive Streamlit dashboard.

🎯 Features
📤 Upload crowd images
👥 Crowd detection using OpenCV (HOG + fallback)
🔥 Heatmap generation for density visualization
📊 Interactive charts (Bar, Pie, Line)
📈 Density estimation metrics
🧠 Intelligent safety recommendations
📄 Downloadable analysis report
🧠 Technology Stack
Python
Streamlit
OpenCV
Pandas
NumPy
Plotly
⚙️ How It Works
Upload a crowd image
System detects people using computer vision
Calculates crowd density
Generates a heatmap for visualization
Displays analytics and charts
Provides safety recommendations based on crowd level
